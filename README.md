# Kelime Kartları APK

Bu repo, HTML kelime kartları uygulamasını GitHub Actions ile Android APK'ya çevirir.

## APK oluşturma

1. GitHub'da yeni bir repo aç.
2. Bu ZIP'in içindeki dosyaları repo'ya yükle.
3. Repo'da **Actions** sekmesine gir.
4. **Build Android APK** workflow'unu çalıştır.
5. İşlem bitince **kelime-kartlari-debug-apk** artifact'ini indir.
6. İçindeki `app-debug.apk` dosyasını telefona kur.

Not: Bu debug APK'dır. Telefonda kurmak için "bilinmeyen kaynaklardan yükleme" izni gerekebilir.
