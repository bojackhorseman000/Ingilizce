# Kelime Kartları APK

Bu repo, kelime kartları HTML uygulamasını GitHub Actions ile Android APK'ya çevirir.

## Kullanım

1. Bu ZIP'i çıkar.
2. İçindeki dosya ve klasörleri GitHub reposunun ana dizinine yükle.
3. GitHub'da **Actions** sekmesine gir.
4. **Build Android APK** workflow'unu çalıştır.
5. İş bitince **Artifacts** bölümünden `kelime-kartlari-debug-apk` dosyasını indir.
6. İçindeki `app-debug.apk` dosyasını telefona kur.

> Not: ZIP dosyasını repo içine tek parça olarak yükleme. GitHub ZIP'i otomatik açmaz.
